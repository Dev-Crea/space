# Space

## Gnome shell extension :
Widget for show disk space using

## Manual Install :
Downloading project
    cd ~/.local/share/gnome-shell/extensions/
    git clone https://github.com/Dev-Crea/space.git
Restart gnome (Alt + F2 -> r) or reboot

## Update schemas
```Linux
glib-compile-schemas schemas/
```
